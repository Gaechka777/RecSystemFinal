В этой директории должны находиться файлы x.npy и y.npy, представляющие из себя предобработанные корректным образом для подачи в модуль BRec данные открытого датасета [Santander](https://www.kaggle.com/competitions/santander-product-recommendation). Эти данные используются для тестирования модуля в режиме default.

Данные для тестирования модели можно найти по ссылкам:

https://drive.google.com/file/d/1tR2DaBxrokUpXvWkv2dzqZBwf7aEAIv6/view?usp=sharing
https://drive.google.com/file/d/1IZZIzsm9k48QytylQTLf5cwekrwHolwk/view?usp=sharing

Для тестирования модели нужно загрузить указанные файлы в папку data локально.
